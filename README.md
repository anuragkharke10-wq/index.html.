# Coffee House

A simple and stylish coffee shop website built using HTML and CSS.

## About the Project
Coffee House is a static website project designed to showcase a modern coffee shop landing page.  
It includes sections for:
- About Us
- Specialities
- Menu
- Contact

## Features
- Responsive navigation bar
- Hero section with background image
- Specialities cards
- Menu section with pricing
- Contact button
- Clean and modern UI

## Tech Stack
- HTML
- CSS
